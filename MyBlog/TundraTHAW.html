006 : Processing raster data in Python with Rasterio

In this tutorial you will learn how to process raster data in python using the popular GIS library rasterio. By the end of this tutorial you will understand how to open raster data in a python environment, create various spectral indiced with band math, generate slope and aspect, create a stacked raster and perform various other image processing tasks.
